Foundations

Core principles of spacing, corner radii, elevation, and borders for a consistent and distraction-free UI.

Spacing System

The 4pt grid system is the backbone of our layout, creating a consistent rhythm and harmonious alignment across all components. All spacing is based on multiples of 4.

--spacing-xs (4px)

For minimal space between inline elements.

--spacing-sm (8px)

Small spacing for tight layouts.

--spacing-md (16px)

Standard spacing between elements.

--spacing-lg (24px)

Large spacing for section separation.

Corner Radii

Radii are used to soften the UI and align with the brand's modern, friendly feel. Consistency is key.

--radius (4px)

--radius-lg (8px)

--radius-xl (12px)

--radius-full

Elevation

A subtle shadow system creates depth and indicates interactivity for floating elements without causing distraction.

--elevation-1

--elevation-2

--elevation-3

--elevation-4

Borders

The standard border style is used for outlining components like input fields or cards, providing clean separation.

--border-default
