Variables and Data Types

Learn the basics of how to store and manage data in Python.

What are variables?

In programming, a variable is a container for storing data values. Think of it like a labeled box where you can put information. In Python, you don't need to explicitly declare the type of a variable; it's determined automatically when you assign a value to it.

Declaring Variables

You create a variable the moment you first assign a value to it. The equals sign = is used to assign values to variables. 

player_name = "Alex"
score = 100
is_active = True

Common Data Types

Variables can store data of different types. Here are some of the most common ones you'll encounter:

  • String (str): Used for text. Enclosed in quotes, like "Hello, Python!".
  • Integer (int): Used for whole numbers, like 42 or -10.
  • Float (float): Used for numbers with a decimal point, like 3.14.
  • Boolean (bool): Represents one of two values: True or False.

Naming Conventions

Following conventions makes your code more readable. In Python, the standard is to use snake_case for variable names. This means all letters are lowercase and words are separated by underscores.
